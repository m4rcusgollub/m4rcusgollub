## 📌 Sobre mim

Sou estudante de programação, focado em desenvolvimento backend e em projetos práticos que unem software e hardware. Estou construindo meu portfólio com aplicações reais, do zero, documentando meu progresso aqui no GitHub.

Gosto de entender como as coisas funcionam por dentro, seja em um script Python, em uma aplicação Java ou em um sensor conectado a um Arduino. Ainda estou no início da jornada, mas com prática constante e vontade de evoluir a cada projeto.
